This repository contains the code for "Jager": a self-piloted, obstacle avoiding robot. The robot itself was built using Arduino and several cheap sensors (a picture is available in "report.pdf"). The project was developed for an Embedded Systems Programming course, in 2016.


The "android" folder contains the android application project;
The "arduino" folder contains the sketch.

install the adafruit library to your Arduino IDE in order for the sketch to compile.
